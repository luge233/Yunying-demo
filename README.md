# 数智生活 · 评论区运营智能体

独立具有运营数据，一键导入评论数据，AI自动分类 + 生成回复建议，可视化仪表盘追踪运营效果。

## 功能

- **📊 运营仪表盘** — 今日新增、待回复、逾期未回、热评率、笔记概览
- **📋 评论列表** — 筛选/排序/批量操作/一键标记已回复
- **🔥 热评识别** — P90阈值 + 笔记内TOP3自动标记
- **🤖 AI智能分类** — 接入大模型（OpenAI兼容API），自动判断评论类型与回复策略
- **💬 AI回复建议** — 流式生成回复文案，支持单条/批量
- **📐 SOP执行清单** — 每日/每周/月度运营清单，进度追踪
- **📝 多方式导入** — JSON文件、表格粘贴、自由文本快速录入

## 使用方法

### 浏览器版（零安装）
直接用浏览器打开 `数智生活·评论区运营智能体.html`。

### 桌面版（.exe）
双击 `数智生活-评论区运营智能体.exe`（由 Python + pywebview 打包）。

## AI功能配置

1. 点击右上角 `···` → `⚙ AI设置`
2. 选择 Provider（DeepSeek / 硅基流动 / 通义千问 / 智谱 / OpenAI）
3. 填入 API Key → 测试连接 → 保存
4. 使用 `🤖 AI智能分类` 和 `💬 批量生成回复`

支持任意 OpenAI 兼容 API。

## 技术栈

- 纯前端：HTML + Vanilla JS + Tailwind CSS + Chart.js
- 数据持久化：localStorage（浏览器版）/ 本地JSON文件（桌面版）
- AI接口：OpenAI 兼容 API（Streaming SSE）
- 桌面打包：Python + pywebview + PyInstaller
